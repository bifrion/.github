## AI Policy
The use of AI assistants is permitted for research and brainstorming. However, you are solely responsible for every line of code you submit. Do not submit code that has been blindly copied from an LLM. If the maintainers finds out that the contribution is complete copy paste of AI generated without sufficient understanding and testing the pull request will be rejected.
Some projects in this organization are reimplementation of existing GPL licensed software. When contributing to these projects, ensure that AI generated code is not derived from the orignal project and not violating their LICENSE terms and conditions.
